## 第0阶段课程

**一般的解决问题过程**：遇见问题上网查,用别人的方法多试几次就解决了。

**劣势**：相同的错误重复再犯，相同的状况重复出现。

**更好的方法**：首先理解应用程序和系统的少数几个基本原理，实战，建立起性能全局观，然后掌握一些必要的性能工具。
